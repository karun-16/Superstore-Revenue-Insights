# Superstore Revenue Insights Analysis

## Overview
This project analyzes superstore sales data to uncover trends, customer behavior, and regional performance using Python data analysis libraries.

---

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Key Features
- Monthly sales trend analysis
- Category-wise sales analysis
- Region-wise sales insights
- Customer segment analysis
- Top-selling products visualization
- KPI summary metrics

---

## Business Insights
- Technology category generated highest sales revenue
- West region achieved strongest sales performance
- Consumer segment contributed highest percentage of sales
- November and December showed peak seasonal sales
